Detailed Description

XTCC Data block. This is an 8 byte block, added to the end of a file created on a non-QDOS/SMSQ system, which records the dataspace requirements for executable files. This, with the correct software on QDOS/SMSQ, allows for the transfer of files between operating systems, without loss of the header data.

Detailed Description

QXL File Header. This structure describes and defines the header part of a QXL Hard drive file. It should always be byte aligned and is 64 + 2 bytes in size. Any other value for the header size is incorrect. The header is actually only 64 bytes long, but the code is appending on the first 16 bit word in the map, for some, as yet, unknown reason. :)

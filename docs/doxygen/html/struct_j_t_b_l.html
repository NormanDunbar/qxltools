Detailed Description

Job Table Entry. This structure defines the layout of a single entry in the QXLtool job table. This is used to determine if the commands typed by the user, at the prompt, are valid, and if so, to call the appropriate function to facilitate the command. Parameters are passed in the usual 'argv' manner, much loved in C programs.

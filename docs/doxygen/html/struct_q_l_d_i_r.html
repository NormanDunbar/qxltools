Detailed Description

QXL Directory Entry. This structure defines the 64 byte (byte aligned) layout of a QXL directory entry. This is a copy of the file header which all files have at the start, although it is never used by the QDOS/SMSQ system. An empty file, in the directory, is always 64 bytes in size. A zero length file is usually a deleted file, especially if the name length is also zero.
